9.5.7 ◐ 重要な限界 e_div は暴落を「予知」しているのか、「反応」しているだけなのか

ここまでの結果は「e_div が高い局面は S&P 500 の下落と重なる」ことを示した。だが 研究の核心的な穴がひとつ残る — 因果の向きだ。

  • 予知 (lead): 下落が始まる前に e_div が立ち上がる → 本物の前兆指標
  • 反応 (lag): 下落と同時 / 後に e_div が立ち上がる → 避けても遅い「初動検知」に過ぎない

これを正面から検証した (scripts/eventstudy_predict_vs_react.py、 出力 data/eventstudy_predict_vs_react.json)。 各下落について「e_div が +0.8σ を最初に超えた日」と「下落開始日」の時間差 (ラグ) を測る。 ラグが負 = 予知、ラグが正 = 反応。

look-ahead は厳密に排除している

e_div の z-score は過去のみの expanding window (最小90サンプル) で計算。 下落の起点は forward 20日リターン (下位10%) で定義するが、これは評価用の ex-post ラベルであり、 e_div の計算には一切入れていない。つまり「未来を知って e_div を作る」不正は無い。

※ 落とし穴の補足: e_div は全期間の約24%で +0.8 を超えている (常時高め)。 そのため「下落の前後60日で最初に +0.8 を超えた日」と素朴に取ると、無関係な過去のエピソードを拾って 見かけ上の『予知』を量産してしまう (実際この素朴版だと中央値 −52営業日という非現実的な値が出た)。 本節は「下落の近傍でアクティブな上昇エピソードの起点」だけを採る方式に修正し、この偽陽性を潰している。

最も公平なテストは、forward リターンで自動検出した起点ではなく、 市場が「ここが天井」と認識する実際の起点で測ること。主要6暴落の結果:

結論: 主要暴落では e_div は「予知」ではなく「反応 (初動検知)」が優勢

真の起点で測ると、4件 (反応) 対 2件 (予知) で反応型が多数。とくに —

  • 最大の暴落 COVID (-33.7%) で e_div は天井時 -0.34 (静か)。下落が始まって4営業日後に立ち上がった。 研究にとって最も都合の悪い事実だが、隠さず示す。
  • SVB・円キャリー・関税の3件も、天井時の e_div は負 or 中立で、立ち上がりは下落開始の8〜19営業日後。 これらは突発的・外因性ショックで、構造の歪みが事前に積み上がるタイプではない。
  • 「予知」に見える2022年の2件は、e_div が起点時点ですでに +2.1 / +1.4 と高かった。 これは「暴落の直前に新しく立ち上がった」のではなく、前から高止まりしていた地合いにたまたま下落が重なったと読むのが正直。 固有の先行シグナルとは言い切れない。

自動検出した40下落イベント全体でも「予知優勢」とは言えない

5y + 20y(OOS) で自動検出した下落を、真の局所ピークを起点に測り、既知6暴落と合算した全40件:

測定方法件数 (cross有)中央値ラグ予知 / 同時 / 反応判定
公平版 (真のピーク起点 + 既知暴落)40 (31)-1 営業日13 / 8 / 10混在 (mixed)
楽観版 (forward-return 起点 = 下落途中)29 (22)-4 営業日14 / 5 / 3予知寄り

「楽観版」が予知寄りに見えるのは定義の罠。forward-return で決めた起点は下落の途中を指すため (実測で局所ピークから平均8〜9営業日後)、e_div が立ち上がる時間的余裕がそのぶん生まれ、人工的に lead が膨らむ。 これを補正した公平版では中央値ラグはほぼ0 (−1営業日)、予知13 : 反応10 でほぼ拮抗 = 混在。

正直なまとめ — リスク管理上の含意

  • 「e_div は暴落を予知する前兆指標」とは言えない。 真の起点で測ると、突発的な大暴落 (COVID・SVB・関税ショック) では下落が始まってから反応している。 「予知」と書けば誇大広告になる。
  • ただし反応でも価値はある。e_div の立ち上がりは下落開始の数営業日後 (中央値 +6) で、 多くの暴落は底まで数週間かかる。初動を検知して残りの下落を回避する「早期警報」としては機能しうる — これは 9.5.6 で VIX に圧勝した事実 (VIX は『暴落後』にしか上がらず後追いになる) とも整合する。 e_div の反応は VIX より速い初動検知であって、未来の予言ではない。
  • 2022年型の「すでに高止まり→下落」パターンは、地合いが悪い局面での持続的な警戒シグナルとして活きる。 ただし常時24%が閾値超えなので、単独では空振り (false positive) も多い。 他指標との併用が前提。

結論を一言で: 「予知ではなく、反応の中でも速い初動検知」。混在だが、突発ショックでは明確に反応側。 リスク管理ツールとしては「避けられる暴落もあるが、最大級の突発暴落 (COVID 型) の入り口は捉えられない」と 弱点まで含めて理解して使うべき。
