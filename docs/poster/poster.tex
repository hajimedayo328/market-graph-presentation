\block{\bt{1.}動機と問い}{%
株価そのものを\textbf{予測}するのではなく、
危険な局面で株を避けて\textbf{\color{accent}下落を回避}できないか。
市場の``つながりの形''から\textbf{2つの指標}を作って調べる。

\vspace{2mm}
\textbf{先行研究の到達点}:
\begin{itemize}
\setlength{\itemsep}{1mm}
\item \textbf{Mantegna (1999)}：相関を距離に変換し、市場を\textbf{ネットワーク}として捉える基礎を作った
\item \textbf{Gidea (2017)}：ネットワークの\textbf{``形''の変化}から危機を捉えられると示した
\item \textbf{Ferreira ら (2021)}：ネットワークの\textbf{``符号''の崩れ}を株式市場で検出した
\end{itemize}
だが\textbf{``形''と``符号''を同時に扱い、両者の関係を調べた研究はない}。

\vspace{2mm}
\textbf{本研究の問い}:
\begin{itemize}
\setlength{\itemsep}{1mm}
\item この\textbf{2つの指標は、たがいに独立}か？
\item その2つを\textbf{組み合わせると下落を回避}できるか？
\end{itemize}

\vspace{2mm}
{\color{muted}\textbf{用語}：\textbf{日次リターン}＝1日の値動き（前日比の変化率）。
\textbf{相関}＝2銘柄の値動きの似方（$+1$＝完全に同調、$0$＝無関係、$-1$＝完全に逆）。}
}

\block{\bt{2.}相関ネットワーク}{%
40銘柄を\textbf{点}、値動きが\textbf{類似した}銘柄どうしを\textbf{線}で結ぶ。
相関 $r$ を距離 $d=\sqrt{2(1-r)}$ に変換し、類似した銘柄を近くに配置する。

\vspace{2mm}
\centering
\includegraphics[width=0.82\linewidth]{../../figs/slide_network.png}\\
{\color{muted}\small 40銘柄の相関ネットワーク（近い＝値動きが類似）}

\vspace{2mm}
\raggedright
このネットワークの\textbf{``形''}と\textbf{``符号''}から2つの指標を取り出す（3.1・3.2）。
}

\block{\bt{3.1}指標1：穴の持続期間}{%
点を近いものから順に結ぶ基準を上げると、\textbf{``輪（穴）''が現れ、やがて消える}。

\vspace{1mm}
\centering
\includegraphics[width=\linewidth]{../../figs/slide_hole.png}

\vspace{1mm}
各穴の\textbf{寿命}（消える基準 $-$ 現れる基準）を測る:

\vspace{1mm}
\includegraphics[width=\linewidth]{../../figs/slide_lifetime.png}

\vspace{1mm}
\raggedright
指標 $L^1$ ＝ すべての穴の寿命の合計:
\[
L^1 = \sum_{(b_k,d_k)\in\mathrm{Dgm}(H_1)}(d_k - b_k)
\]
{\small $\mathrm{Dgm}(H_1)$＝1次元の穴の一覧（$b_k$＝現れる基準、$d_k$＝消える基準）。}
}

% ================= 中列 =================
\column{0.333}

\block{\bt{3.2}指標2：矛盾の数}{%
各線に\textbf{符号}を付ける（正の相関＝$+$、負の相関＝$-$）。
輪を一周して\textbf{符号をかけ合わせる}。

\vspace{2mm}
\centering
\includegraphics[width=0.95\linewidth]{../../figs/slide_balance.png}

\vspace{1mm}
\raggedright
\begin{itemize}
\setlength{\itemsep}{1mm}
\item かけ算が $+$：つじつまが合う（整合）
\item かけ算が $-$：関係がねじれている（\textbf{\color{warn}矛盾}）
\end{itemize}
負の線が奇数個の輪が矛盾。指標 $\nunb$ はその個数:
\[
\prod_{\text{線}\in C}\mathrm{sign}(r) = -1 \Rightarrow \text{矛盾},\quad
\nunb = (\text{矛盾の輪の個数})
\]
}

\block{\bt{3.3}2つの指標の独立性}{%
指標1（穴）と指標2（矛盾）が、たがいにどれくらい似て動くかを\textbf{相関}で測った:

\vspace{1mm}
\centering
\includegraphics[width=\linewidth]{../../figs/slide_scatter_raw.png}

\vspace{1mm}
\raggedright
\textbf{短期（5年）は点がばらける＝\color{accent}独立}（相関 0.15、別々の情報）。
ただし\textbf{データ期間を伸ばすと相関は上がる}:

\vspace{1mm}
\centering
\includegraphics[width=0.8\linewidth]{../../figs/slide_independence.png}

\vspace{1mm}
\raggedright
{\small 長期（20年）は危機を多く含み、両指標が同時に跳ねて\textbf{\color{warn}相関 0.40}（もう独立とは言えない）。}
}

\block{\bt{3.4}2指標の統合：$\ediv$}{%
穴 $L^1$（0〜2）と矛盾 $\nunb$（0〜80）は\textbf{スケールが違う}。
そのまま引くと $\nunb$ が常に勝つので、\textbf{標準化 $z$} で
同じ土俵（過去比の相対位置）に揃えてから引く（$x_t$＝各指標のその日の値）:
\[
z(x_t) = \frac{x_t - \mathrm{mean}(x_{\le t})}{\mathrm{sd}(x_{\le t})}
\]
{\color{muted}\small これで両指標とも平均0・標準偏差1になり\textbf{スケールの差が消える}。mean・sd＝その日までの値（未来は見ない）。}

\vspace{1mm}
差分をとった指標:
\[
\boxed{\ediv = z(\nunb) - z(L^1)}
\]
\begin{itemize}
\setlength{\itemsep}{1.5mm}
\item \textbf{\color{warn}$\ediv > 0$（危険サイン）}：矛盾が穴を上回る＝
市場の関係が\textbf{ねじれ始めている}。まとまり（正相関の群れ）が崩れ、
今まで一緒に動いた銘柄が逆に動く。\textbf{\color{warn}秩序の崩壊の兆し} → 株を避ける
\item \textbf{$\ediv < 0$（平常）}：穴が矛盾を上回る＝
関係のねじれが少なく、\textbf{まとまりが保たれている} → 株を持つ
\end{itemize}
}

% ================= 右列 =================
\column{0.333}

\block{\bt{3.5}なぜ「差分」なのか}{%
片方だけでは効かない。同じ条件（危険サインの日に株を避ける）で
20年運用したときの\textbf{最大下落}:

\vspace{2mm}
\centering
\renewcommand{\arraystretch}{1.3}
\begin{tabular}{lcc}
\toprule
{\bfseries 使う指標} & {\bfseries 最大下落} & {\bfseries シャープ比}\\
\midrule
\textbf{$\ediv$（差分）} & {\color{accent}\textbf{$-20\%$}} & {\color{accent}\textbf{0.76}}\\
矛盾 $\nunb$ だけ & $-45\%$ & 0.47\\
穴 $L^1$ だけ & $-56\%$ & 0.22\\
何もしない & $-57\%$ & 0.48\\
\bottomrule
\end{tabular}

\vspace{2mm}
\raggedright
\begin{itemize}
\setlength{\itemsep}{1mm}
\item 片方だけでは\textbf{何もしないのと大差なし}
\item \textbf{\color{accent}差分にして初めて $-20\%$ に激変}
\end{itemize}
穴を基準に、矛盾の``純粋な超過''を取り出すのが差分の役割。
}

\block{\bt{4.}結果：下落を回避できた}{%
$\ediv>0$ の日に株を売って現金化する戦略を、
\textbf{シミュレーション（過去データでの模擬売買）}で検証:

\vspace{2mm}
\centering
\includegraphics[width=0.92\linewidth]{../../figs/slide_equity_absolute.png}\\[1mm]
{\color{muted}\small \$10{,}000 を5年運用した模擬売買（S\&P 500、取引コスト込み）}

\vspace{1mm}
\raggedright
\begin{itemize}
\setlength{\itemsep}{1mm}
\item \textbf{最大下落が $-25\%\to-17\%$} に浅くなる
\item 20年・16回の検証でも\textbf{ランダム回避の上位1\%}
\item ただし\textbf{リターンは持ち続けに劣る}（下落回避に特化）
\end{itemize}
}

\block{\bt{}今後の取り組み}{%
\begin{itemize}
\setlength{\itemsep}{2mm}
\item \textbf{他市場での再現}（欧州・アジア・暗号資産）で一般性を確かめる
\item \textbf{なぜ差分が効くのか}の理論的な解明（現状は経験的な発見）
\item 位相と符号を統一的に扱う\textbf{数理の枠組み}（コホモロジー等）
\item 暴落の\textbf{予知}への拡張（現状は初動の検知にとどまる）
\end{itemize}
}
