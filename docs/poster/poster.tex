\block{④ 指標① 穴の持続 \textnormal{\small\color{c-muted}Persistent $H_1$}}{
  \fontsize{22}{28}\selectfont

  {\color{c-muted}\fontsize{18}{22}\selectfont
  Vietoris--Rips filtration, $d_{ij}=\sqrt{2(1-r_{ij})}$}

\block{② 手法：2つの位相不変量}{
  \fontsize{22}{28}\selectfont
  同じ相関ネットワークから \textbf{\color{c-accent} 数学的に別種の2つの位相不変量} を同時に抽出。
  \\[3mm]
  \begin{itemize}
    \setlength{\itemsep}{2mm}
    \item \textbf{穴の持続} — Persistent homology $H_1$（$L^1$ ノルム）
    \item \textbf{矛盾の数} — Signed structural balance（frustration）
  \end{itemize}
  \vspace{3mm}
  \[
    \ediv \;=\; z(\nunb) \;-\; z(L^1)
  \]
  {\color{c-muted}\fontsize{18}{22}\selectfont
  組み合わせ指標 $\ediv$ で両者のズレを危険サインとして検出。}
}

\block{⑤ 指標② 矛盾の数 \textnormal{\small\color{c-muted}Signed cycles}}{

  \textbf{\color{c-accent} 負になる三角形} を数える。多いほど符号構造がねじれる。\\[3mm]
  {\color{c-muted}\fontsize{18}{22}\selectfont
  Heider balance / $\mathbb{Z}_2$-cohomology 相当（構造バランス理論）}
