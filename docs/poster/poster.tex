\block{\bnum{1}動機と問題設定}{%
\textbf{\color{accent}株価を「予測」するのではなく、危険な局面で株を避けて「下落を回避」できるか。}
本研究の目的はこれである。

\vspace{2mm}
先行研究の到達点:
\begin{itemize}
\setlength{\itemsep}{1mm}
\item \textbf{Mantegna (1999)} — 相関を距離 $d=\sqrt{2(1-r)}$ に変換し、市場を\textbf{ネットワーク}として捉える基礎を築いた（業種ごとの階層構造を発見）。
\item \textbf{Gidea \& Katz (2017)} — そのネットワークに持続ホモロジーを適用し、構造の``穴''が\textbf{危機の約250日前から増える}＝価格でなく``形''から危機を捉えられると示した。
\item \textbf{Cartwright \& Harary (1956)} — 符号付き関係の``矛盾''を定量化する\textbf{構造バランス理論}。
\end{itemize}

\vspace{1mm}
\textbf{\color{accent}未解決}: 位相（穴）と符号（矛盾）を\textbf{同じ相関ネットワークで同時に}抽出し、両者の関係を検証した研究はない。

\vspace{2mm}
\textbf{本研究の問い}: 40銘柄の日次相関ネットワークから2指標を同時に測り、その独立性と下落回避への有効性を検証する。

\vspace{2mm}
{\color{muted}\small データ: Yahoo Finance (yfinance) から40銘柄（為替・株価指数・個別株・商品・債券・暗号）、2006--2026年、日次終値。}
}

\block{\bnum{2}手法：2つの指標を同時に測る}{%
\textbf{相関ネットワークの作り方}:
40銘柄の日次リターンの相関 $r$ を距離 $d=\sqrt{2(1-r)}$ に変換し、
似た動きの銘柄を近くに配置する。\\[1mm]
そこから\textbf{数学的に別種の2つの指標}を取り出す:
\begin{itemize}
\setlength{\itemsep}{1mm}
\item \textbf{穴の持続期間} $L^1$（つながりの``形''）
\item \textbf{矛盾の数} $\nunb$（符号の``ねじれ''）
\end{itemize}

\vspace{1mm}
\textbf{標準化 $z$ の定義}（各指標を同じ物差しに揃える）:
\[
z(x_t) = \frac{x_t - \mathrm{mean}(x_{\le t})}{\mathrm{sd}(x_{\le t})}
\]
\textbf{過去のデータのみ}を使う（未来を見ない＝先読み防止）。

\vspace{1mm}
\textbf{組み合わせ}:
\[
\boxed{\ediv = z(\nunb) - z(L^1)}
\]
}

\block{\bnum{3}指標1：穴の持続期間}{%
相関ネットワークには``輪（穴）''ができる。
つなぐ距離のしきい値を少しずつ上げると、各穴は\textbf{生まれて、やがて消える}。
\textbf{しきい値を変えても長く残る穴が本物}、すぐ消える穴はノイズ。
\[
L^1 = \sum_{k}(\text{穴}_k \text{の持続期間})
\]

\vspace{2mm}
\centering
\includegraphics[width=0.9\linewidth]{../../figs/slide_network.png}\\[1mm]
{\color{muted}\footnotesize 青=正相関・赤破線=負相関（40銘柄の日次相関ネットワーク）}
}

% ================= 中列（縦に 4→5→6）=================
\column{0.333}

\block{\bnum{4}指標2：矛盾の数}{%
各辺に\textbf{符号}を付ける（正相関 $=+$、負相関 $=-$）。
ネットワーク上の輪（サイクル）を一周し、符号の積を見る:
\[
\prod_{\text{辺} \in C} \mathrm{sign}(r) =
\begin{cases} +1 & \text{整合} \\ -1 & \text{矛盾} \end{cases}
\]
\textbf{負の辺が奇数個}のサイクルが``矛盾''（Cartwright--Harary 1956）。
\[
\nunb = \text{矛盾サイクルの個数}
\]

\vspace{1mm}
{\color{muted}\small 例: 「A と B は同調、B と C は同調、なのに A と C は逆」——この三角形が矛盾。}
}

\block{\bnum{5}独立性 — 短期は成立、長期で崩れる}{%
2指標の相関を\textbf{データ期間別}に計算し直すと:

\vspace{2mm}
\centering
\renewcommand{\arraystretch}{1.2}
\begin{tabular}{lcl}
\toprule
{\bfseries 期間} & {\bfseries 相関} & {\bfseries 判定}\\
\midrule
5年  & 0.15 & ほぼ独立\\
10年 & 0.19 & ほぼ独立\\
15年 & 0.26 & 弱い相関\\
20年 & \textbf{0.40} & {\color{warn}\textbf{独立でない}}\\
\bottomrule
\end{tabular}

\vspace{2mm}
\raggedright
\textbf{\color{warn}正直な開示}: 期間を伸ばすほど相関は単調に上がる。
\textbf{20年では独立とは言えない}。長期は危機（2008・2020等）を多く含み、
危機時は穴も矛盾も同時に跳ねて連動するため。「独立」は\textbf{短期・平常時の近似}。
}

\block{\bnum{6}組み合わせ指標 $\ediv$}{%
2指標を同じ物差しに揃えて差を取る:
\[
\ediv(t) = z(\nunb) - z(L^1)
\]
\textbf{矛盾だけが穴を超えて突出した日} $\Rightarrow$ 危険サイン。

\vspace{1mm}
バックテストでは、$\ediv$ が\textbf{上位20\%の日に株を売って現金化}する
（＝危ない局面を避ける）。予測ではなく\textbf{回避}の指標。
}

% ================= 右列（縦に 7→8→9）=================
\column{0.333}

\block{\bnum{7}なぜ「差分」か — 単独では効かない}{%
同じしきい値（上位20\%で回避）で株を避けたときの\textbf{最大下落}（20年）:

\vspace{2mm}
\centering
\renewcommand{\arraystretch}{1.2}
\begin{tabular}{lcc}
\toprule
{\bfseries 指標} & {\bfseries 最大下落} & {\bfseries シャープ比}\\
\midrule
\textbf{$\ediv$ 差分} & {\color{accent}\textbf{$-20\%$}} & {\color{accent}\textbf{0.76}}\\
矛盾 $\nunb$ 単独 & $-45\%$ & 0.47\\
穴 $L^1$ 単独 & $-56\%$ & 0.22\\
持ち続け & $-57\%$ & 0.48\\
\bottomrule
\end{tabular}

\vspace{2mm}
\raggedright
\textbf{単独では下落を防げない}（穴だけ・矛盾だけでは持ち続けと大差なし）。
\textbf{\color{accent}差分にして初めて $-20\%$ に激変}。
機序: $\ediv$ が高い日は矛盾が穴を超えて突出し、穴を基準に矛盾の``純粋な超過''を取り出す。
}

\block{\bnum{8}結果1：下落を浅くする}{%
\centering
\includegraphics[width=0.98\linewidth]{../../figs/slide_equity_absolute.png}\\[1mm]
{\color{muted}\footnotesize \$10{,}000 を5年運用したバックテスト（S\&P 500、取引コスト込み）}

\vspace{1mm}
\raggedright
$\ediv$ 突出日に株を避けると、\textbf{最大下落が $-25\%\to-17\%$} に浅くなる。
20年の期間外検証（16回）でも最大下落は\textbf{ランダム回避1000本の上位1\%}。\\
{\color{muted}\small ただしリターンは持ち続けに劣り、シャープ比の優位は期間外では消える（下落回避のみが残る）。}
}

\block{\bnum{9}結果2：主因は天然ガス}{%
\centering
\includegraphics[width=0.92\linewidth]{../../figs/slide_source.png}\\[1mm]
{\color{muted}\footnotesize 4つの危機で銘柄を1つずつ抜いた寄与度}

\vspace{1mm}
\raggedright
4つの独立な危機（米関税2025・円キャリー2024・中東2023・ウクライナ2022）で
銘柄を1つずつ抜くと、\textbf{\color{accent}天然ガスが 4/4 で寄与最大}。
銅・中国株が 3/4 で続く。米国株は 1/4 以下。
}

\textbf{\large 検証設計}

\begin{itemize}
\item \textbf{前進検証}: 3年学習/1年テストを20年で16回。しきい値も$z$も学習期間のみから決定（先読み排除）
\item \textbf{ランダム回避との比較}: 同頻度で無作為に回避する1000本より最大下落が浅い（上位1\%）
\item \textbf{因果の向き}: 政策ショック $\to \ediv$ の一方向の関連（グレンジャー因果 $p=0.02$、逆向きは非有意）
\end{itemize}
\end{minipage}
\hfill
\begin{minipage}[t]{0.30\linewidth}
\textbf{\large 結論と限界}

\textbf{結論}: 同じ相関ネットワークから\textbf{\color{accent}短期ではほぼ独立な2指標}（穴・矛盾）を同時に抽出でき、その差分 $\ediv$ は20年の期間外検証で\textbf{下落の浅さに限り}頑健な優位を保つ。

\vspace{1mm}
\textbf{限界}:
\begin{itemize}
\item 独立性は期間依存（20年で相関0.40）
\item 予測でなく初動検知（急落の入口は捉えられない）
\item シャープ比の優位は期間外で消える
\item 差分が効く機序は未解明（危機期間の識別による）
\end{itemize}
\end{minipage}
