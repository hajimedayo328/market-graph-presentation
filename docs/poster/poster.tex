\block{1.\ 動機と問題設定}{%
市場分析の主流は\textbf{価格の予測}だが、本研究は視点を変える。
価格ではなく銘柄どうしの\textbf{\color{accent}連動の構造}を毎日の相関ネットワークにして、
その\textbf{``形''の変化}から構造変化を捉える。

\vspace{4mm}
40銘柄（FX・株指数・個別株・商品・債券・暗号・特殊）の日次相関から
Vietoris--Rips filtration で位相不変量を抽出。

\vspace{4mm}
\textbf{既存研究との位置}: TDA×金融 (Gidea--Katz 2017)、
構造バランス×金融 (Ferreira 2021) など個別には既存。
\textbf{\color{accent}両者を同じネットで同時に扱う先行研究は未見}。
}

\block{4.\ 指標1「穴の持続」}{%
相関ネットに ``輪（穴）'' ができる。
\textbf{基準を変えても残る穴が本物}、すぐ消える穴はノイズ。

\vspace{2mm}
\[
L^1 = \sum_{k} (d_k - b_k),\ (b_k,d_k) \in \mathrm{Dgm}(H_1)
\]

\begin{itemize}
\item 距離 $d_{ij}=\sqrt{2(1-r_{ij})}$ (Mantegna)
\item Vietoris--Rips で $\epsilon$ を上げる filtration
\item $H_1$ サイクルの生死を追い、$L^1$ を日次スカラー化
\end{itemize}

\vspace{2mm}
{\color{muted}\small $\Rightarrow$ 危機時に $L^1$ 上昇（Gidea 2017 と整合）}
}

\block{7.\ 結果1：2指標は無相関}{%
日次 1{,}235 点で穴 $L^1$ と矛盾 $\nunb$ はほぼ無相関 ($r=0.16$)。

\vspace{2mm}
\centering
\includegraphics[width=0.95\linewidth]{../../figs/slide_scatter.png}

\vspace{2mm}
{\color{muted}\small 横=穴、縦=矛盾、日次窓幅30}

\vspace{2mm}
\raggedright
\textbf{結論}: 同じネットから抽出したのに独立
$\Rightarrow$ \textbf{\color{accent}別の情報を持つ2チャンネル}。
}

% ======== 中央列 ========
\column{0.333}

\block{2.\ 手法：2つの位相不変量}{%
同じ相関ネット $G_t$ から\textbf{\color{accent}数学的に別種の2つの位相不変量}を同時抽出。

\vspace{2mm}
\begin{itemize}
\item \textbf{穴の持続} — Persistent $H_1$ の $L^1$ ノルム \emph{(magnitude)}
\item \textbf{矛盾の数} — 符号付きサイクル frustration \emph{(sign)}
\end{itemize}

\vspace{2mm}
独立性を確認した上で標準化差分で組み合わせる:
\[
\boxed{\ediv \;=\; z(\nunb) - z(L^1)}
\]

\begin{center}
{\color{muted}\small $z(\cdot)$: expanding-window z-score (look-ahead 排除)}
\end{center}
}

\block{5.\ 指標2「矛盾の数」}{%
各辺に符号 $\sigma_{ij}=\mathrm{sign}(r_{ij}) \in \{\pm1\}$ を載せ、
独立サイクル $C$ を一周した積で整合／矛盾を判定:

\vspace{-2mm}
\[
\prod_{(i,j)\in C}\!\!\sigma_{ij} = +1\ (\mathrm{balanced}),\ -1\ (\mathrm{unbalanced})
\]
\[
\nunb = |\{C\in\mathcal{B} : \textstyle\prod\sigma = -1\}|
\]

\vspace{2mm}
{\color{muted}\small
Heider (1946) balance / $\mathbb{Z}_2$ 係数コホモロジー $H^1(G;\mathbb{Z}_2)$ 相当。
基底 $\mathcal{B}$ 依存性はあるが balanced/not は基底不変。}
}

\block{8.\ 結果2：主因は天然ガス}{%
4つの独立ショック（米相互関税 2025・円キャリー巻き戻し 2024・
ハマス/イスラエル衝突 2023・ウクライナ侵攻 2022）で
銘柄を leave-one-out で抜くと、
\textbf{\color{accent}NGAS のみ 4/4 で寄与上位}。
COPPER・CHINA50 が 3/4（円キャリーで脱落）、米国株は 1/4 以下。

\centering

}

\block{3.\ 独立性の実証と限界}{%
窓・閾値・市場を振っても $|r|<0.5$、9/10 の設定で $<0.3$。

\vspace{2mm}
\centering
\renewcommand{\arraystretch}{1.3}
\begin{tabular}{lll}
\toprule
{\bfseries 設定軸} & {\bfseries 範囲} & {\bfseries $|r|$}\\
\midrule
Window & 20/30/40/60日 & $\leq 0.23$\\
Threshold & 0.2/0.3/0.4 & $\leq 0.22$\\
Market & USA/EM/CN & 0.16/0.17/0.41\\
\midrule
\multicolumn{2}{l}{全10設定} & max $|r|{=}0.41$\\
\bottomrule
\end{tabular}

\vspace{3mm}
\raggedright
\textbf{正直な開示}: 期間を伸ばすと相関は漸増する
（5年 0.16 $\to$ 10年 0.19 $\to$ 15年 0.26、中国市場 0.41）。
全設定で $|r|<0.5$ だが、``独立'' は\textbf{厳密ではなく近似的}。
}

\block{6.\ 組み合わせ指標 $\ediv$}{%
独立な2チャンネルを同じスケールに揃えて差分:
\[
\ediv(t) = z_t(\nunb) - z_t(L^1)
\]

\textbf{矛盾だけが突出した日}$\Rightarrow$危険サイン。
閾値 $\ediv\geq 0.8$ で株を売って現金化。

\vspace{2mm}
\textbf{因果方向の実証}:
\begin{itemize}
\item Granger 因果: 政策ショック $\to\ediv$, $p=0.022$ (lag=5)。
      逆方向は $p=0.254$ で不成立 $\Rightarrow$ 一方向性
\item cluster permutation で自己相関補正
\end{itemize}

\vspace{1mm}
{\color{muted}\small 注意: $\ediv$ は $L^1$ の低下でも上昇する（純粋な矛盾シグナルではなく2指標の乖離）。}
}

\block{9.\ 結果3：回避戦略の効果}{%
\textbf{予測モデルではなく回避戦略}。
$\ediv$ 突出日に株を売って現金化 $\Rightarrow$
\textbf{最大下落が $-25\%\to -17\%$}（5年 in-sample、コスト込み）。

\vspace{2mm}
\centering
\includegraphics[width=0.98\linewidth]{../../figs/slide_equity_absolute.png}

\vspace{1mm}
{\color{muted}\small \$10{,}000 を5年運用したバックテスト上の資産推移 (S\&P 500)。リターンでは \$1{,}754 負ける。}

\vspace{2mm}
\raggedright
\textbf{20年 walk-forward OOS (16 fold)}: MaxDD $-16\%$ vs B\&H $-34\%$、
同頻度ランダム現金化 500 本に対し\textbf{\color{accent}上位 1\%}。\\
\textbf{正直な開示}: \textbf{Sharpe の優位は OOS で消える}
（$\ediv$ 0.63 $<$ B\&H 0.74）。生き残るのは\textbf{下落の浅さのみ}
$\Rightarrow$ リスク低減の補助指標として位置づけ。
}

\textbf{\large 検証設計}\ {\color{muted}\footnotesize Rigor}

\begin{itemize}
\item \textbf{walk-forward OOS}: 3年学習/1年テスト、20年で 16 fold（look-ahead 排除）
\item \textbf{ランダム現金化 1000 本}: MaxDD $-20\%$ vs 平均 $-42\%$（in-sample 上位 0.1\%、OOS でも上位 1\%）
\item \textbf{VIX ベンチ}: OOS 16 fold で全指標 $\ediv$ 優位（VIX は暴落後の後追い）
\item \textbf{Granger 因果}: 政策ショック $\to\ediv$ は $p=0.022$、逆方向は不成立
\item \textbf{p-hack 反証}: 55 ペア差分の全スキャン。$\ediv$ は政策/構造で上位 22--36\%、戦争では中央値以下 ＝ 都合よく全部上位ではない
\item \textbf{cluster permutation}: 時間集中イベントの自己相関補正
\end{itemize}

\begin{minipage}[t]{0.4\linewidth}
\textbf{\large 結論と限界}\ {\color{muted}\footnotesize Conclusion \& Limitations}

\textbf{結論}: 同じ相関ネットから\textbf{\color{accent}ほぼ独立な2つの位相不変量}を同時に抽出できる。
組み合わせ指標 $\ediv$ は 20年 OOS で\textbf{下落の浅さ（MaxDD）に限り}頑健な優位を保つ。

\vspace{1mm}
\textbf{限界（すべて実証済みの開示）}:
\begin{itemize}
\item 暴落の\textbf{予知ではなく初動検知}（COVID 型の急落の入口は捉えられない）
\item Sharpe・リターンの優位は OOS で消える（B\&H 以下）
\item 独立性は近似的（期間依存で $r$ は 0.16$\to$0.26 に漸増）
\item 矛盾数はサイクル基底に依存（厳密な frustration index は NP 困難）
\end{itemize}

\vspace{1mm}
\textbf{今後}: 他市場での再現。代数トポロジー・cellular sheaf による2指標の理論的統一。
