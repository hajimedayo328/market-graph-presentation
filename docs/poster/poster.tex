\block{① 動機と問題設定}{
  \fontsize{22}{30}\selectfont
  市場分析の主流は\textbf{価格の予測}だが、本研究は視点を変える。
  価格ではなく、銘柄どうしの \textbf{\color{c-accent}連動の構造} を毎日の相関ネットワークにして、
  その\textbf{\color{c-accent}``形''の変化}から構造変化を捉える。\\[3mm]
  40銘柄（FX・株指数・個別株・商品・債券・暗号・特殊）の日次リターン相関を
  Vietoris--Rips filtration で連結性の階層に変換し、位相不変量として観察する。\\[3mm]
  \textbf{既存研究との位置}: TDA×金融は Gidea--Katz (2017) 以降、
  構造バランス×金融は Ferreira (2021)・Nature (2024) 等。
  \textbf{\color{c-accent}両者を同じネットワークで同時に扱う先行研究は未見}。
}

\block{④ 指標① 穴の持続 \textnormal{\small\color{c-muted}Persistent $H_1$}}{
  \fontsize{22}{30}\selectfont
  相関ネットワークにできる ``輪（穴）'' を測る。
  \textbf{基準を変えても残る穴が本物、すぐ消える穴はノイズ}。\\[2mm]
  \[
    L^1 \;=\; \sum_{k} (d_k - b_k), \quad
    (b_k, d_k) \in \mathrm{Dgm}\bigl(H_1(\mathrm{VR}_\epsilon)\bigr)
  \]
  \begin{itemize}
    \setlength{\itemsep}{1mm}
    \item 距離 $d_{ij} = \sqrt{2(1 - r_{ij})}$ (Mantegna)
    \item $\epsilon$ を上げる filtration で $H_1$ サイクルの生死を追う
    \item $L^1$ ノルムを日次スカラー指標として使用
  \end{itemize}
  \vspace{2mm}
  {\color{c-muted}\fontsize{18}{22}\selectfont
  $\Rightarrow$ 危機時に $L^1$ が上昇（Gidea 2017 と整合）。}
}

\block{⑦ 結果① 2指標は無相関 \textnormal{\small\color{c-muted}Independence}}{
  \fontsize{22}{30}\selectfont
  日次1{,}235点の時系列で穴と矛盾はほぼ無相関 ($r{=}0.16$)。\\[2mm]
  \centering
  \includegraphics[width=0.95\linewidth]{../../figs/slide_scatter.png}\\[1mm]
  {\color{c-muted}\fontsize{16}{20}\selectfont
  横軸=穴 $L^1$、縦軸=矛盾 $\nunb$（日次窓幅30）}\\[2mm]
  \raggedright
  \textbf{結論}: 同じネットから抽出したのに独立 $\Rightarrow$
  \textbf{\color{c-accent}別の情報を持つ2チャンネル} として扱える。
}

\block{② 手法：2つの位相不変量を同時実装}{
  \fontsize{22}{30}\selectfont
  同じ相関ネットワーク $G_t$ から、
  \textbf{\color{c-accent}数学的に別種の2つの位相不変量} を同時に抽出する。
  \\[2mm]
  \begin{itemize}
    \setlength{\itemsep}{2mm}
    \item[$\bullet$] \textbf{穴の持続} — Persistent homology $H_1$ の $L^1$ ノルム
                    \hfill \textnormal{\small(magnitude)}
    \item[$\bullet$] \textbf{矛盾の数} — 符号付きサイクル（frustration）
                    \hfill \textnormal{\small(sign)}
  \end{itemize}
  \vspace{2mm}
  独立性を確認した上で、標準化差分で組み合わせる:
  \[
    \boxed{\;\ediv \;=\; z\!\bigl(\nunb\bigr) \;-\; z\!\bigl(L^1\bigr)\;}
  \]
  \vspace{-2mm}
  \begin{center}
  {\color{c-muted}\fontsize{18}{22}\selectfont
  $z(\cdot)$: expanding-window z-score（look-ahead を厳密排除）}
  \end{center}
}

\block{⑤ 指標② 矛盾の数 \textnormal{\small\color{c-muted}Signed cycles}}{
  \fontsize{22}{30}\selectfont
  相関ネットワークの各辺に符号 $\sigma_{ij}=\mathrm{sign}(r_{ij}) \in \{\pm 1\}$ を載せ、
  \textbf{独立サイクル $C$ を一周した積} で整合／矛盾を判定:
  \[
    \prod_{(i,j)\in C} \sigma_{ij} \;=\; +1 \;(\text{balanced}), \;\; -1 \;(\text{unbalanced})
  \]
  \[
    \nunb \;=\; \bigl|\{C \in \mathcal{B} : \textstyle\prod \sigma = -1\}\bigr|
  \]
  \vspace{-1mm}
  {\color{c-muted}\fontsize{18}{22}\selectfont
  Heider (1946) balance / $\mathbb{Z}_2$-cohomology $H^1(G;\mathbb{Z}_2)$ 相当。
  基底 $\mathcal{B}$ 依存性はあるが balanced/not は基底不変。}
}

\block{⑥ 組み合わせ指標 $\ediv$ の使い方}{
  \fontsize{22}{30}\selectfont
  独立な2チャンネルを同じスケールに揃えて差分:
  \[
    \ediv(t) \;=\; z_t(\nunb) - z_t(L^1)
  \]
  \textbf{矛盾だけが突出した日} $\Rightarrow$ 危険サイン。
  バックテストでは閾値 $\ediv \geq 0.8$ で株を売って現金化。\\[2mm]
  \textbf{因果方向の実証}:
  \begin{itemize}
    \setlength{\itemsep}{1mm}
    \item Granger 因果: 政策ショック $\to \ediv$, $p{=}0.022$ (lag=5)
    \item cluster permutation で自己相関補正
    \item walk-forward: 3年学習/1年テスト、$16$ fold
  \end{itemize}
}
