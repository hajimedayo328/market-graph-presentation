## 1. しきい値・パラメータの「なんで?」

### 1-1. 符号の辺は なぜ |r| ≥ 0.3?

**答え**: 0.3が最適だから選んだのではない。**0.2〜0.4のどれで切っても「独立」の結論は変わらない**（むしろ0.4の方が独立性は強く出る）。低すぎるとノイズの辺が入り、高すぎるとサイクルが作れないので、その中間。

| しきい値 | r(L¹, n_unb) | 結論 |
|---|---|---|
| 0.2 | 0.217 | 独立 |
| **0.3（採用）** | **0.162** | 独立 |
| 0.4 | 0.049 | 独立 |

- **コード**: [`compute_gamma_timeseries.py#L34`](../scripts/lib/compute_gamma_timeseries.py#L34) （`threshold: float = 0.3`）
- **データ**: [`independence_robustness.json`](../data/independence_robustness.json) の `threshold_scan`

### 1-2. 窓は なぜ 30日?

**答え**: **30が最適だからではない**。Δσ_ediv は窓60・90の方がむしろ強い。30は「短すぎず反応が遅すぎない中間」として選んだ。独立性の結論は窓20〜60で不変。

| 窓 | r(L¹, n_unb) | Δσ_ediv（イベント反応） |
|---|---|---|
| 20 | 0.234 | 0.59 |
| **30（採用）** | **0.162** | 2.75 |
| 40 | 0.059 | — |
| 60 | −0.063 | **3.43** |
| 90 | — | **3.84** |

- **コード**: [`compute_gamma_timeseries.py#L33`](../scripts/lib/compute_gamma_timeseries.py#L33) （`window: int = 30`）
- **データ**: [`independence_robustness.json`](../data/independence_robustness.json) の `window_scan` ／ [`robustness_window_scan.json`](../data/robustness_window_scan.json) の `summary_table`

### 1-3. 売買しきい値は なぜ e_div ≥ 0.8?

**答え**: **0.5〜1.0のどこで切っても「差分が最良」の順位は不変**。むしろ0.6の方が成績が良く、0.8を選んで最適化したわけではない。**4章の前進検証ではしきい値自体を学習する**（実際の値は0.56〜1.21）ので0.8には依存しない。

| しきい値 | e_div | 不均衡だけ | 穴だけ | 差分が最良? |
|---|---|---|---|---|
| 0.5 | −15% | −38% | −53% | YES |
| 0.6 | **−13%** | −41% | −53% | YES |
| **0.8（3.5表で使用）** | −20% | −45% | −56% | YES |
| 1.0 | −25% | −46% | −54% | YES |
| 1.2 | −35% | −49% | −54% | **NO** |

- **コード**: [`reproduce_poster_numbers.py#L36`](../scripts/reproduce_poster_numbers.py#L36) （`SIGNAL_THRESHOLD = 0.8`）
- **データ**: [`threshold_sensitivity_20y.json`](../data/threshold_sensitivity_20y.json)

### 1-4. 図（2章）の |r| ≥ 0.4 は?

**答え**: **見やすさのためだけ**の値（線が多すぎると絵が潰れる）。指標の計算は0.3で、図とは無関係。実害なし。

- **コード**: [`build_slide_figs.py#L264`](../scripts/build_slide_figs.py#L264) （`TH = 0.4`）

### 1-5. 穴（L¹）のしきい値は?

**答え**: **ない**。半径を0から∞まで動かして全ペアの距離を見るので、切る必要がない。**「穴は閾値フリー、符号は閾値が要る」という非対称性**が、2指標が別物である傍証にもなる。

- **コード**: [`persistent_homology.py#L29`](../scripts/lib/persistent_homology.py#L29) （`correlation_to_distance`）、[`#L39`](../scripts/lib/persistent_homology.py#L39) （`D = np.sqrt(2*(1-R))`）

### 1-6. 取引コスト・ヒステリシスは?

**答え**: 片道0.05%を全エンジンで一貫控除。シグナルは5営業日のヒステリシス（ばたつき防止）。

- **コード**: [`reproduce_poster_numbers.py#L33-34`](../scripts/reproduce_poster_numbers.py#L33)

## 3. 独立性

| 期間 | r(L¹, n_unb) |
|---|---|
| 2006–11年 | 0.45（古いほど連動） |
| 2011–16年 | 0.35 |
| 2016–21年 | 0.22 |
| 2021–26年 | **0.15**（近年ほど独立） |
| 20年通し | 0.40（2008年危機の寄与） |

- 30日おきに間引いても r=0.44（自己相関の影響は小さい）
- 市場別: USA 0.16 / EM 0.17 / **CN 0.41**（中国だけ独立性が弱い ← 正直に認める）
- **コード**: [`compute_gamma_timeseries.py#L85-98`](../scripts/lib/compute_gamma_timeseries.py#L85) （**同じ相関行列**から L¹ と n_unb を両方計算）
- **データ**: [`gamma_timeseries_20y_w30.csv`](../data/gamma_timeseries_20y_w30.csv) ／ [`independence_robustness.json`](../data/independence_robustness.json)

### 3-1. 「独立」の定義を突かれたら

**正直に**: ここで言う独立は **Pearson無相関**の意味。統計的独立の検定（相互情報量・距離相関等）はしていない。実際 Spearman 0.43 > Pearson 0.40 で、**非線形の従属は残っている**。「無相関であって独立ではない」と指摘されたら**認める**のが正しい。

## 4. 手法の「なんで?」

### 4-1. なぜ図の配置に意味がないのか

**答え**: 40銘柄の距離を全部満たす2次元配置は**数学的に存在しない**（3点でも三角不等式が破れうる）。だから配置は諦め、[`spring_layout`](../scripts/build_slide_figs.py#L272)（バネモデル）で見やすく散らしているだけ。**相関の強さは配置に渡っていない**（辺の有無だけ）。負相関の辺も引き合うので「近い＝似ている」は成立しない。

**距離 d=√(2(1−r)) を使うのは図ではなく L¹ の計算の中**。持続ホモロジーは座標を必要とせず、**距離行列だけで穴を計算できる**——これがこの手法を使う理由の1つ。

### 4-2. なぜ差分（引き算）なのか

- **主答（実証）**: 単独指標では効かず、**差分にして初めて改善**（3.5表）
- **補（数学）**: 連動していれば Var[e_div] = 2(1−ρ) → 0 で何も測れない。**独立（0.15）だから超過情報が残る**
- **やっていないこと**: 組み合わせの網羅探索はしていない。変種探索が過剰適合になるのを見たので、**仮説から演繹的に差分を選び、単独指標との比較で検証**した

### 4-3. なぜ三角形でなく長いサイクルを数えるのか

**答え**: 強い相関（|r|≥0.5）だけで見ると、**「正2・負1」の三角形は実データに1つも存在しない**（三角形自体は300個以上あるのに）。市場が株グループ/通貨グループに分かれて構造的均衡が保たれているため、**不均衡は主に長いサイクルで生じる**。

- **コード**: [`homology.py#L138`](../scripts/lib/homology.py#L138) （`signed_cycle_balance`）、[`#L152`](../scripts/lib/homology.py#L152) （`nx.cycle_basis`）
- **正直に**: カウント値は**サイクル基底の取り方に依存**する（L¹ほど数学的にcanonicalではない）

## 射程（弱点を突かれたときの着地点）

> **米国株価指数を中心とした市場では、構造の崩れ（e_div）を使うと下落を有意に回避できる**（ランダム500本中の上位1%、VIXにも勝つ）。
> ただし ①個別の暴落を予知しているわけではない ②指数を抜くと消える ③非米国では効かない ④持ち続けには勝てない。
> だからこれは**普遍的な法則でも予知でもなく、米国市場における「構造の異変の検知力」の実証**。

**先に限定する方が、後から限定させられるより強い。**
