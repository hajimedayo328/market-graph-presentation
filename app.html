🎯 判定の核心: e_div = 2 指標の乖離

判定はたった一つの数字 e_div で行う:

e_div = z(不整合サイクル数) − z(L¹ ノルム)
= z(矛盾の数) − z(穴の総量) = 「符号」と「強さ」のどっちが突出してるか
e_div ≫ 0 (大きく正) 矛盾の数 (符号) が突出 / 穴の総量 (強さ) は平凡 政策ショック型
e_div ≈ 0 (両指標とも平凡) 特に異常なし or 両方が同方向に動く 平常 または 一般ボラ上昇
e_div ≪ 0 (大きく負) 穴の総量 (強さ) が突出 / 矛盾の数 (符号) は平凡 強さ変化型

「他の組み合わせで作った方が良くない？」への補足: L¹ 系 6 個 + n_unb 系 4 個 + balance_rate の計 11 指標で全ペア差分 55 通りを試した。 e_div は trade/structure ショックで上位 22-36%、war では中央値以下 → cherry-pick じゃない (scripts/robustness_ediv_pair_scan.py)。

なぜ「乖離」を見るか: L¹ と n_unb が独立に動くから (corr=0.16)。 同じデータから別の情報を取り出している 2 指標の「ズレ」こそが、 ショックのタイプを識別する鍵。詳細は研究内容タブの Section 8 参照。

🗂 4 つの判定タイプ — どうなったらどの型か

上の e_div の値と L¹ の値を組み合わせて、市場状態を 4 タイプに分類:

政策ショック型 e_div ≥ +0.8
L¹↓ or 横ばい
n_unb↑↑ 突出
e_div+0.8 〜 +1.5+

🗣️ 1 行で言うと: 「あの国とあの国の関係が、政策で逆になっちゃった」

どんなシナリオか: 短期間 (2 週間〜1 ヶ月) に政策イベントが連鎖して、 銘柄ペアの正負関係そのものが反転する状態。 「強さ」は変わらないが「符号」がひっくり返る。

典型例 — 2025-04 Liberation Day 連鎖 (教科書例):

  • 4/2 トランプ「相互関税」発表
  • 4/4 中国「34%報復関税 + 希土類規制」
  • 4/8 トランプ「90日関税停止」
  • 4/9 145%にエスカレート
  • 4/15 Nvidia H20 (AI 半導体) 輸出規制

→ 2 週間で 5 イベント。報復が再応酬を呼ぶ「連鎖性」がカギ。 単発じゃこのタイプにならない (例: 2018-2019 米中貿易戦争は段階的だったので出なかった)。

実際の符号反転 (2025-04 前後):

  • CHINA50 ↔ Dow30: -0.47 → +0.47 (完全逆転)
  • AAPL ↔ ポンドドル: +0.37 → -0.39
  • ドイツ DAX ↔ ドルスイス: -0.33 → +0.30

対応策 (バックテスト S1, look-ahead 完全排除版): 株式ポジション → 現金化 検出後 +57.9% (Buy & Hold +75.4%), MaxDD -17.3% (B&H -25.4%)

強さ変化型 e_div ≤ -0.5
L¹↑↑ 爆発
n_unb軽く動く
e_div-0.5 〜 -1.5

🗣️ 1 行で言うと: 「日経が過去最大下げ」「戦争始まった」「巨大規制 1 個」

どんなシナリオか: 1 回の大きな構造的事件で特定の資産群が一斉に激しく動く。 リスク資産 vs 安全資産のクラスタが鮮明化。 でも「誰と誰の関係」は変わらない (符号は安定)。連鎖ではなく単発。

典型例 — 3 つのパターン:

  • 市場構造ショック: 2024-08-05 円キャリー巻き戻し (日経過去最大下げ)、 2025-08-01 サマー・ボラショック ← 強制ポジション解消
  • 戦争・地政学: 2023-10-07 ハマス・イスラエル戦争、 2024-04-13 イラン・イスラエル攻撃 ← 安全資産買い + リスク資産売り
  • 中規模一方的政策: 2022-08 米 CHIPS 法、 2024-05 バイデン対中 EV 100% 関税、EU EV 関税 ← 単発の規制 (連鎖がない)

→ 共通点は「単発」「強さ (相関絶対値) が一気に変わる」。 連鎖性がないので符号反転は起こりにくい。

対応策 (バックテスト S2 long, look-ahead 完全排除版): 逆張りで買い増し シャープレシオ +0.70 (B&H +0.71 と同等), MaxDD -20.9% (B&H -25.4%)

一般ボラ上昇 -0.5 < e_div < +0.8 かつ L¹ ≥ 0.7
L¹↑ (やや上)
n_unb↑ (やや上)
e_div≈ 0

🗣️ 1 行で言うと: 「市場全体が今日は荒れてる、VIX 高い」

どんなシナリオか: 突発的なニュースで市場全体が一時的に騒がしくなる状態。 銘柄が一斉に大きく動くが、構造的な変化はない。 「誰と誰が仲がいいか」は変わらず、ボラ (変動の激しさ) だけ上がる。

典型例:

  • FOMC サプライズ (予想外の利上げ・利下げペース変更)
  • 雇用統計サプライズ (予想と大きく外れる)
  • 四半期決算ラッシュ (Mag5 の決算が偏る時期)
  • VIX スパイク全般 (突発的なテールイベント)
  • トランプの突発ツイート系 (短期影響)

→ 「リスクオフ」で株売られて国債買われる、みたいな普段の流れ。 関係性は維持されてる。L¹ も n_unb も似たような幅で動くので e_div の差が出ない。

実証 (VIX 自動スパイク 25 件): L¹ Δσ = +0.23、n_unb Δσ = +0.28、e_div ≈ +0.09 (ゼロ近辺)

対応策: 通常のリスク管理 (ストップロス、ポジションサイズ調整)

平常 上記いずれにも該当しない
L¹平均近辺
n_unb平均近辺
e_div-0.5 〜 +0.8

🗣️ 1 行で言うと: 「普段の市場、特に何もない」

どんなシナリオか: 両指標とも平常域。ネットワーク構造に特異な変化なし。 5 年間の約 6 割はこの状態。これが市場のデフォルト。

典型例:

  • 大きなイベントなし、トレンド継続中
  • レンジ相場・横ばい局面
  • 夏枯れ・年末年始のような流動性薄期
  • 決算シーズン以外の平常週

対応策: Buy & Hold で OK / 通常運用

過去 5 年の 3 つの数字を、上下 3 段に分けて重ねて表示する:

3バックテスト結果 — 実際に投資に役立つか?

バックテストとは何か (1 行で): 「もし過去にこの指標を見て自動売買してたら、どれくらい儲かってたか」を 過去データでシミュレーションすること。

📊 バックテスト 3 視点サマリー (1 枚)

📌 何を発見したか — 4 つだけ
  1. 5〜20 年どの期間でも、ただ持ってるより効率よく稼げる (グラフ 1: 赤い棒 = この戦略、灰色棒 = ただ持ってるだけ)。20 年で差が最大。
  2. 暴落の時 (2008 / 2020) の損失が半分で済む (グラフ 2: 灰色は 20 年で 57% 沈むが、赤は 20% で踏み止まる)。
  3. 株指数 (SP500 など) を抜くと戦略が完全に効かなくなる (グラフ 3: INDEX 抜きだけ短い赤)。
  4. 銘柄を 40 から減らすと「異変は察知できる、でも稼げない」 (グラフ 4: N=40 だけ高く、N=35 以下でガクッと落ちる)。 = 「シグナル察知の力」と「稼ぐ力」は別物。

用語: Sharpe = 効率 (リスクに対する儲けの良さ、高いほど良い) / MaxDD = 最大の下落幅 (一番沈んだ深さ、小さいほど良い) / B&H = ただ持ってるだけ (比較基準)

📋 戦略の中身 — S1〜S6 + 方向

各戦略は「ある条件 (シグナル) を満たす日」を検知して売買する。 条件を満たした日 = "シグナル ON" の日。

戦略シグナル条件意味
S1: e_div_highe_div ≥ +0.8 「政策ショック型」を検知。関税系の連鎖を捉える。
S2: e_div_lowe_div ≤ -0.5 「強さ変化型」を検知。円キャリー巻き戻し等。
S3: abs_zL1|z(L¹)| ≥ 1.0 L¹ が極端 (上 or 下 1σ 超) の日。「強さ」単独で動いた日。
S4: abs_zunb|z(n_unb)| ≥ 1.0 不整合サイクル数が極端な日。「符号」単独で動いた日。
S5: zL1 OR zunbz(L¹) ≥ 1.0 OR z(n_unb) ≥ 1.0 どちらか片方が上振れした日 (OR ロジック)。広めに検知。
S6: zL1 AND zunbz(L¹) ≥ 1.0 AND z(n_unb) ≥ 1.0 両方が同時に上振れした日 (AND ロジック)。非常に稀な異常日 (2.9% のみ)。

各戦略を 2 方向で実装:

  • _short: シグナル ON の日に 株を売って現金化。シグナル OFF で買い戻し。 → 「警報サインで逃げる」戦略
  • _long: シグナル ON の日に 株を持つ。シグナル OFF で売却 (現金)。 → 「警報サインで買う」逆張り戦略

💡 主な結論 (このバックテストから)

  • S1 short (e_div ≥ 0.8 で現金化) は Sharpe +0.88 で B&H (+0.71) 上回り: リターン自体は +57.9% と Buy & Hold (+75%) を下回るが、 最大ドローダウンが -25.4% → -17.3% に大幅改善。 「政策ショック検知でリスクオフ」によりリスク調整後リターンで優位。
  • S2 long (e_div ≤ -0.5 で保有) は Sharpe +0.70 (B&H +0.71 と同等): 強さ変化型ショックが過ぎた後の方が S&P500 がよく上がるという、逆張り効果。 最大 DD は -20.9% (B&H -25.4% より浅い)。
  • S6 short (両指標 AND で現金化) は発火率でも +76.5%: 非常に稀な「両指標同時異常」を捉えると、Sharpe +0.76 と高水準。
  • L¹ や n_unb の絶対値だけを見る S3/S4 は Buy & Hold に勝てない。 e_div (乖離) を見ることに意味がある。

⚠️ Limitation (正直に):
・取引コスト 0.05%/片道で計算 (より厳しめのフルコストでは差が出る可能性)
・スリッページなし (実際は約定価格がズレる)
・シグナル翌日寄付き約定 + z-score は過去のみ expanding window → look-ahead bias は完全排除済み (v2)
・S&P500 のみ (他資産では結果が違う可能性)
・過去最適化リスク: 閾値 0.8 / -0.5 は過去データから決めた値。 本当に未来でも効くかは out-of-sample 評価が必要。
実運用化前にこれらを全部潰す必要がある。現段階では「過去に対しては効いた」段階。

📊 検証の堅牢性 — Walk-forward OOS + 多市場再現性

In-sample (期間全体で最適化) の数字は「過去に対して効いた」だけ。 未来でも効くかを確認するため、walk-forward Out-of-Sample (OOS) と別市場・別期間での再現性を独立に検証。

Walk-forward OOS (3 年学習 → 1 年テストの転がし) — look-ahead 完全排除版

期間fold 数OOS Sharpe勝率 (vs B&H)代表的な fold
10y (2016-2026)6+0.451/6 (17%) 2023 fold Sharpe +1.70 (24-06)、2022 fold +0.78
15y (2011-2026)11+0.543/11 (27%) 2023 fold Sharpe +2.23 (24-07)、2016 fold +1.37
20y (2009-2025)16+0.653/16 (19%) 最高 fold Sharpe +2.23 / +1.82 / +1.22

→ z-score look-ahead を完全排除した結果、In-sample Sharpe は +1.16 → +0.88、 Walk-forward OOS Sharpe も 10y +0.45 / 15y +0.54 とやや低下したが、 MaxDD (-17%) が B&H (-25%) を大きく下回り、リスク管理ツールとしての価値は健在。
→ Bull market では機会損失が出るが、bear / volatile 年に α が出る傾向。 リターン狙いではなくリスク管理ツールとして位置付けるのが妥当。

多市場再現性 (5 期間 × 3 市場)

同じパイプラインを別市場・別期間に当てても、主要発見が再現することを確認:

  • USA 5y / 10y / 15y / 20y データで L¹ と n_unb の独立性 (corr=0.16-0.40) 再現
  • 新興国 (EM) 5y データで同じ独立性パターン再現
  • 中国 A 株 5y データで「市場構造ショック → L¹」が再現 (Δσ=+1.47, p=0.007)
  • USA 関税ショック (n=11 イベント) で「n_unb のみ反応」を再現 (Δσ=+0.38, p=0.018)

→ 単一市場・単一期間の偶然ではなく、異なる市場で繰り返し観察されるパターン。
→ 追い風: Wang & Xu (2025) が独立に「2024 年米国対中関税で中国 LSCBM が 3.6 倍急増」を報告 → 我々の trade_policy 発見と別角度から相互補強している。

⚠️ 正直な制約 (Limitation)

  1. trade_policy の強反応は 2025-04 Liberation Day cluster が大きく寄与している (単一イベント駆動の側面)
  2. OOS Sharpe +0.45 〜 +0.54 (in-sample +0.88 からの低下、いずれも look-ahead 完全排除版) → 過去最適化バイアスを認容している
  3. 現在は Vantage デモ口座 + 単一ブローカー (CFD SP500.r) で paper trading 検証中。 ライブ資金での運用は future work
  4. 4 手法の (α) limit (関手の合成) では trade_policy 前兆が消失 → 手法選択依存性が残る
  5. 因果性は仮説段階。「政策イベント → 符号反転」の方向性は permutation test で支持されているが、 因果推論としての厳密な証明はまだ

📅 運用フロー (日次)

  1. 毎朝 JST 07:00 過ぎ にこのページをチェック (VPS が日次自動更新)
  2. 📡 「最新の市場状態」カードを見て、判定を確認
  3. 判定に応じてポジション調整:
    • 政策ショック型 → SPY (or 国内では JPY 建て指数 ETF) を売却 / ショート
    • 強さ変化型 → 逆張りで買い増し (S2 long 戦略、Sharpe +0.70〜+0.96・期間で変動)
    • 一般ボラ上昇 → 通常リスク管理 (ストップ強化、ポジション縮小)
    • 平常 → Buy & Hold 継続
  4. 判定が変わったらポジション切替 (前日と判定が変わってない日は何もしなくて OK)
  5. 記録: 判定 / 売買額 / 翌日のリターンを Notion or Excel で記録
